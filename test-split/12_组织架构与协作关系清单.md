# 组织架构与协作关系清单

## 1. 公司组织架构及主要职责

### 1.1 总裁办

**总裁办职责：** 根据公司战略发展规划、经营目标、规章制度和重大事项的决策权、运营权、管理权。
**具体角色：** 总裁办负责公司战略规划、市场拓展、客户关系维护、跨部门协调等全面管理。

### 1.2 客户部

**客户部职责：** 负责客户管理、商机跟进、合同签订、回款管理、客户维护、客户分析、市场拓展。
**具体角色：** 客户部负责市场拓展（负责市场拓展目标制定、推广活动管理、客户资源分配）、客户经理（负责客户开发、销售推进、合同签订）

### 1.3 产品研发部

**产品研发部职责：** 负责产品规划、小程序研发、产品版本迭代、系统测试、BUG修复、功能优化、产品维护等。
**具体角色：** 产品经理（负责产品规划、需求分析、版本管理等）、研发工程师（负责产品开发、功能实现、接口开发）

### 1.4 综合服务部

**综合服务部职责：** 负责客户综合服务支持、客服系统运营、FAQ库维护、培训支持、客户投诉处理。
**具体角色：** 客服经理（负责客服团队管理、客服流程优化、客服培训）、客服专员（一线客户咨询、问题处理、投诉跟进）

### 1.5 财务部

**财务部职责：** 负责财务核算、预算管理、差旅报销管理、薪酬福利、税务合规、费用控制等。
**具体角色：** 财务经理（负责财务部门管理、预算制定、费用审批）

### 1.6 综合管理部

**综合管理部职责：** 负责行政管理、人力资源招聘、入职离职管理、绩效考核制度制定等、行政组织协调、IT运维及信息安全等。
**具体角色：** 行政专员（负责行政事务、办公采购、后勤保障）、人事专员（负责人员招聘、培训计划制定）

### 1.7 采购部

**采购部职责：** 负责供应商管理、合同签订、供应商维护、合同合规、验收付款、采购归档等。
**具体角色：** 采购专员（负责采购申请、供应商管理、合同跟进）

## 2. 日常协作关系清单

1. 新员工入职、转正、培训、离职相关事务 → 综合管理部
2. 差旅申请、预算审批、费用报销 → 综合管理部
3. 采购申请、合同签订、供应商对接 → 采购部+总裁办
4. IT账号、设备、VPN等问题报备 → IT专员
5. 客户商机跟进、合同签订、客户投诉 → 综合服务部
6. 产品功能开发、版本迭代、合同需求 → 产品经理+研发团队
7. 综合客服、问题处理、客户投诉 → 综合服务部
8. 公司重大事项决策、费用审批 → 总裁办

## 3. 跨部门协作说明

日常跨部门事务由协作部门直接对接完成，由对应职能部门提供支持；如需跨部门协作，由双线领导协调确定，由信息部门统一上报总裁办。特殊事项需直接对接对应职能接口，确保流程顺畅，信息同步及时。
