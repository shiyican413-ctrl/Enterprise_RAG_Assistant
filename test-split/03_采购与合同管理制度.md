# 采购与合同管理制度

## 1. 采购申请流程

公司在采购设备等物品前需进行采购申请，先批准后执行，先审批后采购。采购需填写采购申请单，内容包括：采购物品、数量、规格、型号、预算金额等。采购申请需与业务需求一致，预算审批需与财务审批结合。采购申请未经批准不得实施，采购前需经过审批。

## 2. 供应商管理

供应商信息需详细记录，包括：供应商资质、联系方式、产品质量、价格性价比等信息。对供应商进行综合评估，包括价格、质量、交货期、售后服务等方面，供应商信息将在公司内部系统中定期更新。

## 3. 合同签订流程

所有采购项目金额超过5000元的需签订书面形式的合同。合同需明确双方权利义务、价格、交货期限、违约责任等条款；需标明双方和合同合法性、责任豁免条款、权利义务等；需由直属领导审批确认后方可签字盖章。

## 4. 付款节点

采购项目的付款方式参照预算审批、预付款项、验收合格后尾款流程执行：10%预付款、30%进度款、60%验收款。一般项目执行3-6-1付款模式（30%预付款、60%验收款、10%尾款）。需按照合同约定执行预付款、进度款、尾款等各阶段付款要求。

## 5. 采购验收

采购设备或服务交付完成后，由采购人员填写验收单。采购负责人需在验收单上签字确认，验收不合格者记录在案。合同签订需确保验收条款明确，验收单需与合同内容一致。验收不合格者将追究违约责任。

## 6. 归档要求

所有采购及合同相关文件统一归档管理，包括采购申请单、供应商资质、合同原件、验收单、发票等资料。相关文件由行政部门统一保管。合同保管期限不少于3年，重大项目不少于10年，确保可追溯。
