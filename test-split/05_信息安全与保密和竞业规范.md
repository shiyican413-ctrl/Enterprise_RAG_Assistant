# 信息安全与保密和竞业规范

## 1. 商业数据管理

公司商业数据按级别分为：机密数据、保密文件、核心数据、商业来源、经营策略等，其中核心数据需最高级别保护。一般数据按照公开、内部、保密、机密等分级，员工需根据数据级别申请相应权限后查看使用，防止私自泄露、复制、转发。

## 2. 客户信息管理

客户信息包括客户信息、联系方式、成交记录、项目方案等，存放在公司内部系统。员工仅可在业务范围内查阅相关客户信息，禁止私自查询、下载、外发非授权客户数据。客户信息不应用于与公司业务无关的用途，禁止私自存储于私人设备。

## 3. 权限最小化

公司各系统权限、数据、设备权限严格执行权限最小化原则，划分员工可访问、不可访问的数据范围。员工权限按岗位变动时调整原有权限和新增权限，员工离职需注销全部权限。

## 4. 泄密风险防控

如发现公司内部文件、数据、合同、客户信息被泄密，需通过邮件、企业微信等途径立即上报，并注明泄密内容、途径、影响等。信息安全部门通过相应渠道防控泄密风险，追究当事人责任。

## 5. 账号安全规范

员工应将所有账号实行一人一账号，专人专用，禁止账号混用、转借。需设置复杂密码，禁止使用简单密码。员工需在办公设备上设置开机密码，离职时需主动退出系统。

## 6. 违规处置

违反信息安全规范，包括：泄露公司数据、越权访问、盗用账号、泄密信息等行为，视为严重违纪，给予内部警告、降级、解除劳动合同等处分。造成公司经济损失的，需承担赔偿责任。
