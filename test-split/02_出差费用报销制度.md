# 出差费用报销制度

## 1. 出差项目费用标准

### 1.1 住宿标准

员工出差住宿费用实行分级报销：总监级及以上为高标准，经理级为中标准，普通员工为基础标准。住宿标准需结合当地市场行情和公司财务规定综合考虑。

### 1.2 餐费补贴

餐费补贴包括：午餐、晚餐、宵夜等费用，按出差天数和地区差异计算，参照公司差旅费用报销标准执行。出差期间员工可自行选择餐厅就餐，补贴以实际消费为准。

### 1.3 交通费用

出差交通包括：机票、火车票、高铁票、出租车、地铁等费用。员工需自行选择经济实惠的出行方式，结合最优成本选择最合适的交通方式。

### 1.4 住宿分级

按出差地的城市级别设定住宿分级标准：一线城市标准最高，二线城市次之，三线城市标准最低。同城同级别员工享受相同住宿标准。

## 2. 报销票据要求

所有报销票据必须为正规增值税发票，抬头必须为真实有效的公司全称，税号必须准确。票据需完整保存并按公司财务要求提交报销。

## 3. 报销申请流程

员工需在出差结束后5个工作日内提交报销申请，提交OA系统申请流程，由直属领导审核确认后提交财务部门复核。票据审核需与出差申请一致。

## 4. 报销时间规定

报销申请每月1-15日、16-30日分批处理。员工需在规定时间内完成报销申请，超出规定时间者将收取相应滞纳金。如因特殊原因无法在规定时间内完成报销，需提前3个工作日向财务部门申请延期。

## 5. 报销违规处理

如有虚假报销、虚假行程等违规行为，一经发现将严肃处理。同一个项目不得重复申请报销。发票内容与实际不符者将被驳回并要求重报。
